# 标签相关快捷键

```
新建标签：Cmd + t

关闭标签：Cmd + w

切换标签：Cmd + 数字 || Cmd + 左右方向键

切换全屏：Cmd + enter

```

# 分屏相关快捷键

```

垂直分屏：Cmd + d

水平分屏：Cmd + shift + d

切换屏幕：Cmd + option + 方向键 || Cmd + [ 或 Cmd + ]


```
