iTerm2

标签相关快捷键

新建标签：Cmd + t

关闭标签：Cmd + w

切换标签：Cmd + 数字 || Cmd + 左右方向键

切换全屏：Cmd + enter

分屏相关快捷键

垂直分屏：Cmd + d

水平分屏：Cmd + shift + d

切换屏幕：Cmd + option + 方向键 || Cmd + [ 或 Cmd + ]

光标控制

Cmd + return 全屏
Cmd + / 光标位置
Cmd + r 只是换到新一屏，不会像 clear 一样创建一个空屏
ctrl + u 清除当前行
ctrl + a 到行首
ctrl + e 到行尾
ctrl + w 删除光标之前的单词
ctrl + k 删除到文本末尾

快速选择文本

Cmd + F, Tab

粘贴板历史记录

Cmd + Shift + H

操作快照

Cmd + Option + B
